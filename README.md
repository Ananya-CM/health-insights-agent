# 🏥 Health Insights Agent

An AI-powered web application that analyzes blood test reports and provides personalized health insights using large language models.

> **Disclaimer:** This tool is for educational and informational purposes only. It does not replace professional medical advice.

---

## ✨ Features

- 📄 Upload blood test PDFs or use a built-in sample report
- 🤖 AI-driven health analysis using Groq-hosted LLaMA models
- 💬 Follow-up Q&A on your report using RAG (retrieval-augmented generation)
- 🔐 User authentication via Supabase
- 💾 Persistent chat session history
- 🔁 Automatic model fallback (4-tier: LLaMA 4 → LLaMA 3.3 → LLaMA 3.1 → LLaMA3)

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Streamlit |
| LLM API | Groq (LLaMA models) |
| Auth & DB | Supabase (PostgreSQL) |
| PDF Processing | PDFPlumber |
| RAG | LangChain + FAISS + HuggingFace Embeddings |

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/health-insights-agent.git
cd health-insights-agent
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Set up Supabase

- Create a project at [supabase.com](https://supabase.com)
- Run `public/db/schema.sql` in your Supabase SQL Editor
- Copy your **Project URL** and **anon key**

### 4. Get a Groq API key

- Sign up at [console.groq.com](https://console.groq.com)
- Create an API key

### 5. Configure secrets

Create `.streamlit/secrets.toml` (use the provided example file):

```toml
GROQ_API_KEY = "your_groq_api_key"
SUPABASE_URL = "your_supabase_url"
SUPABASE_KEY = "your_supabase_anon_key"
```

### 6. Run the app

```bash
streamlit run src/main.py
```
